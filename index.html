Not-A-Modal

Kinda, sorta, not really modals.

For when a modal won't due but you need to show lots of content on top of existing content...

Download on GitHub

Or keep reading to learn more...

How it Works

You can use links or buttons to toggle not-a-modals.

Just add the .modal class and a data-target that matches the ID of element you'd like to toggle. Modal content should be wrapped in a <div> with the .modal-menu class.

Launch Modal 1

Some Title ×

Some content...

Some Other Title ×

Some other content...

<a class="modal" data-target="#modal1" href="#">Launch Modal 1</a>

<div class="modal-menu" id="modal1">
    <div class="container">
        <h2>Some Title <a class="close modal-close" href="#">×</a></h2>
        <p>Some content...</p>
        <button class="btn btn-small modal-close">Close</button>
    </div>
</div>

Don't Forget Fallbacks

Because Not-A-Modal is powered by JavaScript, it should only be used for non-essential content (if there is such a thing), or content that can be accessed in an alternate way.

When active, the default link behavior is turned off. You should probably include an href value as a fallback for when JavaScript is not enabled.

See this in action at PAWS New England.

Requires jQuery

Not-A-Modal requires jQuery.

Obviously, you need to include not-a-modal.js, too.

<script src="http://code.jquery.com/jquery-1.9.0.min.js"></script>
<script src="not-a-modal.js"></script>
